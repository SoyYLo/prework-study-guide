Prework Study Guide

HTML

  • Hypertext Markup Language
  • Builds the foundation or structure of the code
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Cascading Style Sheets
  • Defines the style of the webapp
  • Like decorating the house

Git

  • Version Control system
  • Software on your local computer that tracks changes you make to your code
  • Source Code Management
  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch & switches to it

JavaScript

  • Programming Language
  • Makes the website user-friendly & interactive
  • Like opening the house door or cooking in the kitchen
